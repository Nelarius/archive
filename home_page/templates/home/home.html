Research

Physics

Dislocation mechanisms on near surface voids I performed molecular dynamics simulations on near surface voids under a tensile stress in order to study the hypothesis that dislocations may be responsible for the onset of small surface protrusions. For more information, check out my Master's thesis


Software

Here is some of the stuff that I've worked on.

wrenly C++ binding generator for the Wren programming language. The goal is to generate bindings for wrapping C++ types in a foreign class, and wrapping class methods and free functions in order to implement foreign methods. Source code is available here.

morphoviewer.js A javascript library for loading and displaying 3d models, in a number of different formats. The library can display various surface properties, such as the surface curvature and surface orientation. Source code available on Github.

secon An agent-based economy simulation that I wrote in Lua, based on the work of Steiglitz, Honig, and Cohen and Doran and Parberry. Source code available on Github. A presentation that I gave on this project (in finnish) is available here.
